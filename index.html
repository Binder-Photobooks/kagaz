Bulk printing discount

PDF Manager

Every Photobook and Trade Book order renders as two separate files: the wrap cover (back cover · spine · front cover, spine width computed from page count) and the book block (interior single pages, not spreads). Art Prints render as one 4-up artboard file. All at 300 DPI with 0.1″ bleed and vector trim marks. Render now generates a real, downloadable PDF right here in the browser — good for proofing and for many print shops. It's raster page content rather than fully prepress-grade vector text, so a server render worker (Puppeteer/similar) still re-renders each file with true vector text and CMYK conversion before a large run ships; that stage isn't built yet. Rendered files live only in this browser tab's memory — re-render after a refresh.
