Bulk printing discount

Order 10 or more copies of the same book and every copy drops to the bulk rate automatically — no code needed. That's 50% off the per-copy price on Photobooks and 40% off on Trade Books, applied the moment you set quantity to 10+ at checkout. Extra-page charges still apply per copy at the same per-page rate.

Need more than 50 copies, or a custom trim size? Contact us for a bulk quote.

PDF Manager

Every Photobook and Trade Book order renders as two separate files: the wrap cover (back cover · spine · front cover, spine width computed from page count) and the book block (interior single pages, not spreads). Art Prints render as one 4-up artboard file. All at 300 DPI with trim marks and bleed, text kept as vector. Generating the actual files requires a server-side render worker — this panel is the admin-facing queue and download point; see the note at the end of this build for the rendering pipeline.
